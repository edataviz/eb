Home screen

This is the main screen that opens after successful login. Here, you can press the button to open the Dashboard, or click the shortcut icons to quickly open the corresponding functions. These shortcuts can be added, deleted or reordered. To do that click on the button.
  • Config Shortcuts
    • Add shortcut
    • Delete shortcut
    • Move shortcut

Config Shortcuts

When you press the Config Shortcuts button, the screen will go into shortcut setting mode. Each shortcut icon will now have a dashed border. At the same time, the Done button and instructions appear next to it.

After making changes to shortcuts, press the Done button to complete.

Add shortcut

In shortcut setting mode, you move the mouse to the main menu and click on any function, that function will be added to the end of the shortcut list.

Delete shortcut

In shortcut setting mode, move the mouse over the shortcut you want to delete, then an X icon appears in the upper right corner of the shortcut, click on it to delete the shortcut.

Move shortcut

In shortcut setup mode, you can drag and drop the shortcut to other locations.

File functions

Open an existing diagram

Click button Open  in the toolbar then select a diagram in the list.

Create a new diagram

Click button New  in the toolbar. The diagram name will be reset to [Untitled Diagram] and all current diagram content will be cleared. If you want to get back the contents please click button Undo  in the toolbar.

Named or rename a diagram

Click button Rename  in the toolbar and input the name for diagram.

Save diagram

Click button Save  in the toolbar. To save diagram as a copy, click the arrow down button, then select Save as new diagram

Editor functions

Insert a item to diagram

Drag-drop or click an item from the panel at the left side to insert it into the editting diagram.

Insert an image

Click icon Plus at the end of the toolbar and choose Insert Image. Then you can specify the URL of image or browse for image in local computer to upload. 

Connecting items

Move the mouse over the source item, the connection points would be shown. Drag a point and drop onto the target object.

   

Basic item editting

Select an item by click on it, the handles will be displayed. Then you can drag these handles to resize or rotate the item.

                       

You can change the item's apperance by using format panel at the right side, includes fill color, border (line) color, border style, text color, text size and many more options.

To change the item's text, there are several ways:

  • Double click the item
  • Select item then press Enter or F2
  • Right item then choose Edit in the popup menu

  

Edit item data

Each item can be set extended properties. To do that, right item and choose Edit Data... You can enter property name, click Add Property, then input value. Click Apply button to complete.

The container

There is a special item called container that can contain other items. You can drag items from outside into the container or move out. It can be collapsed / expand to hide / show it's items.

The toolbar

Click the icon View  then you can show/hide Format panel, Outline window, Layers window.

Other actions:

Zoom in/out

Undo / Redo

Delete item

Bring item to front / Send to back

Change item fill color / line color / shadow

And more ...

File functions

Open existing chart

Select chart group and select a chart from dropdown in the toolbar.

Create new chart

Click button New in the toolbar. The current chart settings will be clear so please make sure your chart is saved.

Chart settings

Basic settings

In the Chart tab, you can set the titles and their color and some other options:

Column stacking: When this is set, all the item with type of column will be stacked.

Vertical grid lines: show / hide vertical grid line of the chart.

Legend: show / hide chart legend.

Size: resize the height of the chart. This is just for viewing in the Advance Graph screen, not affected in the Dashboard chart display.

Time range

The time range (period) of the chart is set ending by the selected date in the toolbar and beginning by the Time range setting backward.

Auto refresh (realtime chart)

A chart can be auto updated with latest data, often be used with tags to get data from external data system (like PI, IP21, Exaquantum ...). This has two parameters: time range and refresh cycle (Auto refresh).

When Auto refresh is Disabled, the chart will be static, to reload data you can click button Refresh chart in the toolbar. Otherwise, chart will check and load new data after each refresh cycle time.

Chart items

The Items panel display all chart items. Click on each item to select it and control it's configurations.

Add new chart item

Click button Add chart item, a item will be appended to the end of the items list and will be current selected item.

Select a chart item

Click an item in the list to select it. The selected chart item will have light yellow background color.

Delete a chart item

Move the mouse over a chart item, the X sign appears at the right side, click the X to remove chart item.

Chart item data

To config what data would be displayed in chart item, click button Config.

Select Facility -> Type -> Object -> Data source -> Attribute, select other needed options (eg. Event type, Flow phase) then click button Append to expression, and click button Apply to complete.

Multiple y-axis

Chart can have multiple y-axis. You can config a chart item to use a specific y-axis by set it's title, color and position (left or right side).

Chart item style

Title: set item's title

Color: click to set item's color

Type: set item's visual type (line, column, area ... )

Dash: set item's line style (none (solid), dash, dot ... )

Marker:

  • Point (default): display each data point as a circle
  • Value: Display circle and value on each data point
  • No: hide point circle and value

Suffix: set the suffix for displaying value in data point tooltip



Config Dashboard

The Config Dashboard function allows users to create and setup dashboard screens that contain diagram objects, data label, charts, and gauges. To check Dashboard display please open screen Dashboard from menu Data Visualization.

Config Dashboard use all features of Diagram function. Please refer to Diagram help to know the basic guidelines.
  • Dashboard items
    • Chart
    • Gauge
    • Data value
    • Fit dashboard

Dashboard items

Chart

To add a chart to dashboard, drag a chart item from left panel to the editor.

Set chart for item

The chart dashboard item should be set a chart that was created in Advanced Graph function. Right click the item and select Config, then select a chart in the config box and click button Apply.

 

Gauge

To add a gauge to dashboard, drag a gauge item from left panel to the editor.

Right click the gauge item, choose Config to config the gauge.

To hide the red-band, set Red-band min and max value to same as Value min

To config data for displaying by the gauge, right click gauge item and choose Expression. Select the Facility -> Type -> Object -> Data source -> Attribute, click button Append to expression. Click button Apply to complete.

Data value

To add a data-value to dashboard, drag data-value item from left panel to the editor.

Right click data-value item and choose Expression to config displaying data. This setting is the same as for gauge above.

You can directly edit the expression to set the text for displaying content. Example:
Value 1: <span style='color:red'>{ENERGY_UNIT_DATA_VALUE.EU_DATA_GRS_VOL.703.1.1}</span>

Fit dashboard

To auto resize dashboard to fit in the Dashboard screen, tick the Fit on loaded option.



Dashboard

The dashboard business function allows users to create data visualization screens that contain diagram objects, data label, charts, and gauges.

To create Dashboard please open screen Config Dashboard from menu Config
  • Toolbar
    • Change dashboard
    • Auto update
    • Edit
    • Refresh

Toolbar

Change dashboard

Click the dashboard name to open dropdown and select another dashboard to use.

Auto update

When enabled, the dashboard will automatically reload latest data after a specific time that set in dashboard configurations. When disabled, date selector will be shown to allow you to select a date then click button Refresh to load data manually.

Edit

Click button Edit to open current dashboard in the Config Dashboard screen to edit.

Refresh

Click Refresh button to reload dashboard data.
